# 红点清除计划 H5 小游戏

这是一个可直接运行的手机端 Canvas H5 小游戏。

## 直接打开
双击 `index.html` 可直接在浏览器打开。

## 本地预览服务器
如果希望用本地服务预览，在装有 Node.js 的电脑上运行：

```bash
node dev-server.cjs
```

然后打开：

```text
http://127.0.0.1:5178
```

## 文件说明
- `index.html`：页面入口
- `styles.css`：页面样式
- `game.js`：游戏逻辑
- `assets/`：背景、键盘、球、障碍物等切图资产
- `dev-server.cjs`：轻量本地预览服务器
